Esta aplicación tiene como objetivo facilitar el redimensionamiento y optimización de imágenes. Su interfaz está prácticamente toda en la misma ventana y las opciones son simples. La interfaz está dividida en 6 partes, indicadas por un número del 1 al 6. Solo necesitas encontrar la que te interesa.

1- Caja de archivos

Suelta los archivos de imágenes que deseas optimizar allí dentro o haz clic en el botón para cargar archivos usando el cuadro de diálogo estándar de tu sistema.


2- Ajustes de las imágenes

Son 5 ajustes posibles descritos a continuación:

  • Compresión

    Define de dónde vendrá la compresión de las imágenes.

    • Tinify cuando sea posible: Usará siempre el servicio de Tinipng si está en línea, tiene una clave de API y el tipo de imagen es compatible. De lo contrario, será local.
    • Mejor resultado: Disparará 2 rutinas de compresión, una para Tinypng y otra para local, siempre que Tinypng cumpla con los requisitos anteriores. De lo contrario, será solo local. Al final, se usará la imagen con el menor tamaño.
    • Siempre local: Usará siempre el motor interno de la aplicación para realizar todas las actividades.

  • Color de fondo

    Define el color de fondo para conversiones a formatos sin canal alfa (transparencia). Actualmente solo JPG. Elige el color transparente (ajedrezado) para aprovechar el canal alfa. Si el color elegido es transparente y la imagen de destino tiene canal alfa y se convierte a JPG, se usará negro como color de fondo.


  • Conversión

    Define el formato al cual se convertirá la imagen resultante. Si no se elige ninguno, la imagen permanece en su formato original. Las excepciones son BMP, PCX y HEIC, que se convierten automáticamente a otro formato según la configuración si no se elige un formato específico en este ajuste.


  • Archivos

    Define cómo se creará la nueva imagen en archivo. Las opciones son:

    • Sobrescribir: El archivo original será sobrescrito.
    • Crear carpeta: Se creará una carpeta llamada optimizada, si aún no existe, y el archivo se creará en ella con el nombre original. Si ya existe un archivo con el mismo nombre en esa carpeta, será sobrescrito.
    • Nuevo archivo: Se creará un nuevo archivo con el sufijo "-optimizado".
    • Copia de seguridad: Hace una copia de seguridad del archivo original con el sufijo "-backup" antes de sobrescribirlo.

  • Dimensiones

    Define las dimensiones de la imagen que se está procesando.

    • No alterar: La imagen no sufrirá ningún redimensionamiento.
    • Escalar: La imagen se redimensiona en uno de los ejes. O ancho o alto. La elección aparece cuando esta opción está seleccionada. El nuevo tamaño en píxeles y a qué eje se refiere. El tamaño del otro eje se calcula de forma que la imagen mantenga la misma proporción.
    • Ajustar: La imagen se redimensionará para caber dentro de un rectángulo cuyas dimensiones, ancho y alto, se solicitan cuando esta opción está seleccionada. El nuevo tamaño será el mayor posible para caber en las dimensiones elegidas.
    • Cubrir: La imagen se redimensionará para cubrir el rectángulo cuyas dimensiones, ancho y alto, se solicitan cuando esta opción está seleccionada. El nuevo tamaño será el menor posible para cubrir las dimensiones elegidas. Cuando se usa en Tinypng, se utiliza IA para determinar dónde cortar la imagen.
    • Miniatura: En local es lo mismo que Cubrir. En Tinypng es capaz de alargar los fondos para crear miniaturas. Como en cubrir, se solicitan ancho y alto.


3- Barra de Comandos

Los 3 primeros botones permiten marcar y desmarcar opciones referentes a metadatos de la imagen. Son, en orden: Información de copyright, de geolocalización y fecha de la imagen.

Los 3 últimos, también en orden, cambian el idioma de la interfaz, hacen una donación al autor e invocan la pantalla de Configuraciones.


4- Contadores

Las optimizaciones hechas por esta aplicación se contabilizan en este espacio. Tanto vía Tinypng como local. Se mantienen un total general y otro mensual.

El conteo mensual tanto local como online (Tinify) es solo de las compresiones hechas usando esta app. Cada imagen cuenta como una compresión independiente de haber conversiones y/o redimensionamientos. Para ver cuántas compresiones hiciste en Tinify en el mes corriente habilita "Mostrar el conteo mensual de compresiones de Tinify" en configuraciones.


5- Totales

Muestra el total ahorrado y el número de optimizaciones hechas. El porcentaje se refiere al total ahorrado comparado con el tamaño antes de las optimizaciones.
