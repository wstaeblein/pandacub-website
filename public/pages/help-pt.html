Interface Principal

Este app tem por objetivo facilitar o redimensionamento e otimização de imagens. Sua interface está praticamente toda na mesma janela e as opções são simples. A interface está dividida em 6 partes, indicadas por um número de 1 a 6. Basta achar a que lhe interessa.


1- Box de arquivos

Arraste arquivos de imagens suportadas sobre o box para processa-las. O app irá indicar aceitação animando as bordas. Solte os arquivos de imagens que deseja otimizar ali dentro ou, alternativamente, clique o botão para carregar arquivos usando a caixa de diálogo padrão do seu sistema. Arquivos que não são de imagens ou de formatos não suportados serão mostrados com uma mensagem de erro na próxima tela mas o processamento não será interrompido.


2- Ajustes das imagens

São 5 ajustes possíveis descritos a seguir:

  • Compressão

    Define de onde virá a compressão das imagens.

    • Tinify quando possível: Usará sempre o serviço da Tinipng se estiver online, tiver uma chave de api e o tipo de imagem for suportado. Caso contrário será local.
    • Melhor resultado: Irá disparar 2 rotinas de compressão, uma para o Tinypng e outra para o local, desde que o Tinypng cumpra os requisitos acima. Caso contrário será somente local. No fim a imagem com o menor tamanho será usada.
    • Sempre local: Usará sempre o motor interno do aplicativo para fazer todas as atividades.

  • Cor de fundo

    Define a cor de fundo para conversões para formatos sem canal alpha (transparência). Atualmente apenas JPG. Escolha a cor transparente (xadrez) para usufruir do canal alpha. Se a cor escolhida for transparente e a imagem destino tiver canal alpha e for convertida para JPG, preto será usado como cor de fundo.


  • Conversão

    Define o formato para o qual a imagem resultante será convertida. Se nenhum for escolhido a imagem fica no seu formato original. As exceções são BMP, PCX e HEIC que são automaticamente convertidas para outro formato conforme configuração se um formato especifíco não for escolhido neste ajuste.


  • Arquivos

    Define como a nova imagem será criada em arquivo. As opções são:

    • Sobrescrever: O arquivo original será sobrescrito.
    • Criar pasta: Uma pasta chamada otimizada será criada, se já não existir, e o arquivo será criado nela com o nome original. Se um arquivo de mesmo nome já existir nessa pasta, ele será sobrescrito.
    • Novo arquivo: Um novo arquivo com o sufixo "-otimizado" será criado.
    • Cópia de segurança: Faz uma cópia de segurança do arquivo original com o sufixo "-backup" antes de sobrescreve-lo.

  • Dimensôes

    Define as dimensôes da imagem sendo processada.

    • Não alterar: A imagem não sofrerá nenhum redimensionamento.
    • Escalar: A imagem é redimensionada em um dos eixos. Ou largura ou altura. A escolha aparece quando esta opção está selecionada. O novo tamanho em pixels e a que eixo se refere. O tamanho do outro eixo é calculado de forma a deixar a imagem com a mesma proporção.
    • Caber: A imagem será redimensionada para caber dentro de um retângulo cujas dimensões, largura e altura, são pedidas quando esta opção está selecionada. O novo tamanho será o maior possível para caber nas dimensões escolhidas.
    • Cobrir: A imagem será redimensionada para cobrir o retângulo cujas dimensões, largura e altura, são pedidas quando esta opção está selecionada. O novo tamanho será o menor possível para cobrir as dimensões escolhidas. Quando usado no Tinypng IA é usada para determinar onde cortar a imagem.
    • Miniatura: No local é o mesmo que Cobrir. No tinypng é capaz de alongar os fundos para criar miniaturas. Como o cobrir, largura e altura são pedidos.


3- Barra de Comandos

Os 3 primeiros botões permitem marcar e desmarcar opções referentes a metadados da imagem. São eles em ordem: Informação de copyright, de geolocalização e data da imagem.

Os 3 últimos, também na ordem, troca a língua da interface, faz uma doação ao autor e invoca a tela de Configurações.


4- Contadores

As otimizações feitas por esse aplicativo são contabilizadas nesse espaço. Tanto via Tinypng como local. São mantidos um total geral e outro mensal.

A contagem mensal tanto local como online (Tinify) é apenas das compressões feitas usando este app. Cada imagem conta como uma compressão independente de haver conversões e/ou redimensionamentos. Para ver quantas compressões você fez no Tinify no mês corrente habilite "Mostrar a contagem mensal de compressões do Tinify" em configurações.


5- Totais

Mostra o total economizado e o número de otimizações feitas. A porcentagem é referente ao total economizado se comparado ao tamanho antes das otimizações.


6- Rodapé

Aqui temos um indicador de conexão com a internet, que tenta aferir em tempo real se essa conexão existe. Se não existir todas as atividades serão via o motor interno do aplicativo (local).

Temos também um menu com 4 links:

  • Privacidade: Invoca página com a nossa política de privacidade.
  • EULA: Invoca página com a nossa licença de uso.
  • Sobre: Invoca box sobre este aplicativo e seu autor. Aqui você encontra contatos, outros aplicativos e dados diversos.
  • Ajuda: Esta página.

Dicas & Observações

  • Arquivos comprimidos que forem para a pasta otimizada usarão a pasta se ela já existir e sobrescreverão arquivos com o mesmo nome dentro da pasta.
  • Arquivos comprimidos com sufixo, receberão o sufixo mesmo se mudarem de extensão (ex. estão mudando de formato tb).
  • Auto orientar resolve problemas como o de imagens que estão de cabeça pra baixo mas tem um EXIF tipo: Orientation: Rotate 180. Este comando vai seguir o EXIF sempre que houver. Note que nem sempre imagens assim aparecem de cabeça pra baixo. Qualquer software que leve este EXIF em consideração ira mostrar a imagem corretamente, mas nem todos são assim.
  • Tocar o som ao terminar o processamento é útil se vc colocar varias imagens para processar e for fazer outra coisa. Assim o som te avisa que o processamento terminou.
  • A contagem mensal tanto local como online (Tinypng) é apenas das compressões feitas usando este app. Cada imagem conta como uma compressão independente de haverem conversões e/ou redimensionamentos. Para ver realmente quanto você gastou da sua quota mensal no Tinypng, habilite "Mostrar a contagem mensal de compressões do Tinypng".
  • Quando o arquivo de origem é um SVG, não é computada economia pq não há como comparar imagem vetorial com raster e pelo motivo da conversão nunca ser gerar imagens menores. A economia só é computada entre imagens raster.
  • Cor de fundo pode ser usada para:
    • Transformar o fundo transparente de uma imagem num fundo opaco da cor passada.
    • Fornecer uma cor de fundo para uma conversão de imagem com transparência (alpha channel) para uma que não tem. Se não for fornecido (mantido como transparente) será usado preto (RGB 0, 0, 0).
  • Não é possível substituir o fundo opaco de uma imagem por outro de cor diferente.


Interface de Resultados

Depois que imagens são arrastadas ou abertas e o processo de compressão, redimensionamento e conversão é iniciado a tela abaixo aparece. Nela você apenas acompanha este processo. A interface está dividida em 6 partes, indicadas por um número de 1 a 6. Basta achar a que lhe interessa.


1- Lista de imagens processadas

Contém as imagens sendo processadas para a sua conferência e acompanhamento. Inclui os seguintes campos:

  • Uma miniatura, que mostra muitas vezes somente o centro da imagem original.
  • O nome do arquivo.
  • Indicador de conversão com o formato origem e destino.
  • Um ícone designando se o processamento é local ou remoto.
  • Tamanho original, comprimido e porcentagem de compressão.
  • Tempo gasto no processamento.
  • Status do processo:
    • Processando - animação
    • Completo com êxito - ícone verde
    • Um erro ocorreu - ícone vermelho

2- Indicador de processamento

Indica visualmente e por porcentagem o quanto do processamento já foi concluído.


3- Contador de imagens

Indica quantas imagens estão sendo processadas e quantas já terminaram.


4- Tempo de processamento

Indica o tempo total do processsamento de todas as imagens.


5- Ações

Neste momento apenas o botão de voltar a interface principal. Nada impede que você arraste novas imagens a serem processadas sobre esta interface de resultado. O app irá indicar a aceitação e processa-las do mesmo jeito.


6- Indicador de processamento do Tinypng.com

Se houver uma chave de API válida e estiver habilitado nas configurações, mostra quantas compressões foram feitas no mês com essa chave. Assim fica fácil você controlar quantas compressões já fez.



Configurações

Ajuste aqui as configurações do aplicativo. A interface está dividida em 4 partes, indicadas por um número de 1 a 4. Basta achar a que lhe interessa.


1- Chave de API do Tinypng

Se optar por abrir uma conta no Tinypng, para ter acesso a sua poderosa API de compressão e redimensionamento de imagens, entre sua chave de API nesta caixa de texto.


2- Conversões forçadas

Os formatos BMP, PCX e HEIC são suportados apenas para leitura. Portanto, para serem comprimidos e/ou redimensionados devem obrigatoriamente ir para outro formato. Aqui você escolhe esse formato padrão. Note que quando optar por um outro formato de conversão no seletor da interface principal, ele terá prioridade sobre esta configuração.


3- Outras configurações

  • Sempre no topo - Mantem a janela do aplicativo sempre sobre as outras.
  • Sinal sonoro: Emite um sinal sonoro ao fim do processamento.
  • Contagem mensal do Tinypng: Mostra na interface quantas compressões foram feitas usando o serviço Tinypng no mês corrente.
  • Correção de orientação: Tenta corrigir a orientação de uma imagem com base no seu EXIF, caso exista e só em formatos suportados.
  • Zerar todos os contadores: Zera as contagens de compressões feitas pelo aplicativo e começa tudo do zero novamente.

4- Botões de comando

Salva as configurações ou volta sem salvar.
