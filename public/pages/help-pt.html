Este app tem por objetivo facilitar o redimensionamento e otimizaçào de imagens. Sua interface está praticamente toda na mesma janela e as opções são simples. A interface está dividida em 6 partes, indicadas por um número de 1 a 6. Basta achar a que lhe interessa.

1- Box de arquivos

Solte os arquivos de imagens que deseja otimizar ali dentro ou clique o botão para carregar arquivos usando a caixa de diálogo padrão do seu sistema.


2- Ajustes das imagens

São 5 ajustes possíveis descritos a seguir:

  • Compressão

    Define de onde virá a compressão das imagens.

    • Tinify quando possível: Usará sempre o serviço da Tinipng se estiver online, tiver uma chave de api e o tipo de imagem for suportado. Caso contrário será local.
    • Melhor resultado: Irá disparar 2 rotinas de compressão, uma para o Tinypng e outra para o local, desde que o Tinypng cumpra os requisitos acima. Caso contrário será somente local. No fim a imagem com o menor tamanho será
    • Sempre local: Usará sempre o motor interno do aplicativo para fazer todas as atividades.

  • Cor de fundo

    Define a cor de fundo para conversões para formatos sem canal alpha (transparência). Atualmente apenas JPG. Escolha a cor transparente (xadrez) para usufruir do canal alpha. Se a cor escolhida for transparente e a imagem destino tiver canal alpha e for convertida para JPG, preto será usado como cor de fundo.


  • Conversão

    Define o formato para o qual a imagem resultante será convertida. Se nenhum for escolhido a imagem fica no seu formato original. As exceções são BMP, PCX e HEIC que são automaticamente convertidas para outro formato conforme configuração se um formato especifíco não for escolhido neste ajuste.


  • Arquivos

    Define como a nova imagem será criada em arquivo. As opções são:

    • Sobrescrever: O arquivo original será sobrescrito.
    • Criar pasta: Uma pasta chamada otimizada será criada, se já não existir, e o arquivo será criado nela com o nome original. Se um arquivo de mesmo nome já existir nessa pasta, ele será sobrescrito.
    • Novo arquivo: Um novo arquivo com o sufixo "-otimizado" será criado.
    • Cópia de segurança: Faz uma cópia de segurança do arquivo original com o sufixo "-backup" antes de sobrescreve-lo.

  • Tamanho

    Define o tamanho da imagem sendo processada.

    • Não alterar: A imagem não sofrerá nenhum redimensionamento.
    • Escalar: A imagem é redimensionada em um dos eixos. Ou largura ou altura. A escolha aparece quando esta opção está selecionada. O novo tamanho em pixels e a que eixo se refere. O tamanho do outro eixo é calculado de forma a deixar a imagem com a mesma proporção.
    • Caber: A imagem será redimensionada para caber dentro de um retângulo cujas dimensões, largura e altura, são pedidas quando esta opção está selecionada. O novo tamanho será o maior possível para caber nas dimensões escolhidas.
    • Cobrir: A imagem será redimensionada para cobrir o retângulo cujas dimensões, largura e altura, são pedidas quando esta opção está selecionada. O novo tamanho será o menor possível para cobrir as dimensões escolhidas. Quando usado no Tinypng IA é usada para determinar onde cortar a imagem.
    • Miniatura: No local é o mesmo que Cobrir. No tinypng é capaz de alongar os fundos para criar miniaturas. Como o cobrir, largura e altura são pedidos.


3- Barra de Comandos

Os 3 primeiros botões permitem marcar e desmarcar opções referentes a metadados da imagem. São eles em ordem: Informação de copyright, de geolocalização e data da imagem.

Os 3 últimos, também na ordem, trocam a língua da interface, faz uma doação ao autor e invoca a tela de Configurações.


4- Contadores

As otimizações feitas por esse aplicativo são contabilizadas nesse espaço. Tanto via Tinypng como local. São mantidos um total geral e outro mensal.

A contagem mensal tanto local como online (Tinify) é apenas das compressões feitas usando este app. Cada imagem conta como uma compressão independente de haver conversões e/ou redimensionamentos. Para ver quantas compressões você fez no Tinify no mês corrente habilite "Mostrar a contagem mensal de compressões do Tinify" em configurações.


5- Totais

Mostra o total economizado e o número de otimizações feitas. A porcentagem é referente ao total economizado se comparado ao tamanho antes das otimizações.
